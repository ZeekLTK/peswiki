General

  • Special::Options - WoaS options
  • Special::Search - search all wiki pages

Lists

  • File::, Image:: - pages in the embedded namespaces
  • Locked::, Unlocked:: - encrypted/plain pages
  • Special::All Pages - user created pages
  • Special::Dead Pages - nonexistent pages that other pages link to
  • Special::Orphaned Pages - pages that no other page links to
  • Special::Recent Changes - recently modified pages
  • Tagged::, Untagged:: - tagged/untagged pages

Page commands

  • Special::Delete Page - delete a specific page
  • Special::Duplicate Page - duplicate a specific page
  • Special::Go to - go to a specific page
  • Special::New Page - create a new page

Special features

  • WoaS::Aliases - view/edit page title aliases
  • WoaS::CSS::Core, WoaS::CSS::Custom - view the core CSS and edit the custom CSS
  • WoaS::Hotkeys - view/edit WoaS access keys and default key bindings
  • WoaS::Plugins - enable/disable and view/edit WoaS Javascript plugins

Wiki maintenance

  • Special::Import - import wiki from a WoaS html file (including earlier versions)
  • Special::ImportWSIF - import a WSIF file containing one or more wiki pages
  • Special::Export - export wiki into multiple linked static XHTML files
  • Special::ExportWSIF - export wiki into multiple or single WSIF files
  • Special::Erase Wiki - delete ALL wiki pages and reset with default data
