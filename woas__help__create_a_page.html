  1. Access Special::New Page
  2. Type the page title. Maximum length is 256 characters, although you might want to use a shorter title to not interfere with the navigation bar. Page titles are case sensitive, all characters are allowed except angled brackets, square brackets, curly braces, and pipe, hash, or double-quote characters (<>[]{}|#")
  3. Press the Enter key or click the OK button
  4. When asked if you'd like to add a link into the main menu, accept by pressing the Enter key or clicking the OK button, or refuse by pressing the Escape key or clicking the Cancel button
  5. Edit the contents of the main text area by typing normally. You can freely change the default text (a level one heading with the page name)
  6. Use the editor's buttons to insert wiki syntax to format the text or type the wiki syntax manually (plain text does not need any wiki syntax)
  7. Save your changes

Create a missing page

An alternate method of creating a page is to type a new page title inside square brackets and then attempt to access the unexisting page.

Example:
[[My new page]]
Is rendered as:
My new page

Troubleshooting

There are some issues which could prevent you from creating a page.
  • You didn't type a title for the page in the input box that popped up
  • There is already a page by that title
  • The wiki is being run in read-only mode, so no changes are allowed. See the Wiki is read-only page for instructions on how to return the wiki to editing mode
  • You are trying to use a namespace that is reserved by WoaS. In this case a message will pop up letting you know the namespace is reserved and giving you an opportunity to choose a different title/namespace for the page
