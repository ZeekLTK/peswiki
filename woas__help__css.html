The look and feel of WoaS can be customized by specifying custom CSS in the WoaS::CSS::Custom reserved page. You can use WoaS core CSS definitions available in WoaS::CSS::Core as a reference when you are writing CSS customization.

NOTE: A tutorial about CSS is available at http://www.w3schools.com/css/.

Customizing CSS

  1. Access WoaS::CSS::Core either directly or from Special::Advanced
  2. Copy the CSS definitions you want to customize
  3. Access WoaS::CSS::Custom either directly or from Special::Advanced
  4. Edit the page and paste the classes and ids you copied in previous step
  5. Define the rules for the CSS classes and ids.
  6. Save your changes by clicking the Save icon in the navigation bar.
All changes will be visible immediately.

Restore the default CSS settings

You can always restore WoaS default CSS by deleting the content of WoaS::CSS::Custom.

Import CSS rules into your wiki

You can import the CSS rules from another copy of WoaS by following the instructions for importing another wiki into the current one and placing a check in the box next to Import custom CSS.

Importing from older versions of WoaS can garble the layout of the wiki. There are two ways to avoid this:
  1. Manually transfer the CSS from the older version of WoaS into the WoaS::CSS::Custom page by copying and pasting it, checking the class and id of each entry against the default CSS rules on the WoaS::CSS::Core page to verify that it's still in use before saving your changes.
  2. Create a new page with style tags in it and paste your CSS into that page to test it before adding it to the WoaS::CSS::Custom page.

Create a test page for your custom CSS

  1. Access Special::Options either directly or from Special::Advanced
  2. Make sure that the box is not checked next to Remember current page and navigation history. Save the change if you had to uncheck the box.
  3. Create a new page
  4. Insert this opening tag at the top of the page:
    <style type="text/CSS">
  5. Paste your CSS rules beneath the tag.
  6. Insert this closing tag at the bottom of the page:
    </style>
  7. Save changes
Once you have tested your CSS on a single page you can choose whether to move such CSS definitions to WoaS::CSS::Custom or not.
