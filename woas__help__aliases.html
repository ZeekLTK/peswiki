Aliases are replaceable shortcuts used in wiki links.

Example

If $JD is defined in WoaS::Aliases as John Doe then the link [[$JD::Bio]] will be converted to [[John Doe::Bio]] when rendered by WoaS.

Aliases can turn out to be particularly useful to shorten typing of long page names.

Add an alias

An alias is added by editing the WoaS::Aliases reserved page and adding an alias line to the page. For example:

$JD John Doe

$JD is the alias, which will be replaced when found in wiki page title references, and "John Doe" is the text which will replace $JD. You can specify any part of page titles in aliases, like namespaces or page prefixes/suffixes.
