All user-created pages (and some which are part of reserved namespaces) can be edited.

If a page is editable the Edit icon will be available in the navigation bar; you start editing the page by clicking such icon. The user interface will switch to an editor-mode set of buttons and edit areas.

Editing

In order to edit pages, click the Edit icon in the navigation bar. You can find all the user pages by accessing Special::All Pages directly or either from Special::Advanced.
  1. Click the Edit icon in the navigation bar or double-click on the page if you've enabled that setting in the wiki Options.

  2. Edit the contents of the main text area by typing normally.

  3. Use the editor's buttons to insert code to format the text, or type the code in manually. Note that the use of plain text doesn't require any code.

  4. Save your changes by clicking on the Save icon in the navigation bar.
