This wiki offers two ways to search for content:
  • The search box in the menu which uses the WoaS::Template::Search page as a template.
  • The Special::Search page, which you can get to by clicking the Advanced icon in the menu and looking in the Pages section.
It is recommended that you do one-word searches to avoid lengthy searches that pop up unresponsive script warnings. Also, type only the exact search term(s) into the box without surrounding the term(s) with quotes or using other characters unless the content you're searching for contains them.

The results display links to pages with the search term in their title followed by links to pages that contain the search term, the number of hits, and the the highlighted search term surrounded by content.

Add a search box or search link to a page

The search box in the menu can be deleted by editing the menu and removing the [[Include::WoaS::Template::Search]] line. If you would like a search box in another page in the wiki, add the line you removed from the menu where you would like the box to appear in the page. To add a link to the Special::Search page to any page in the wiki, insert [[Special::Search]] anywhere in the page.
