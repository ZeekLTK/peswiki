You can define key bindings for hotkeys (also called 'access keys') by editing the WoaS::Hotkeys page.

Example:
$SAVE s
$EDIT e
The $SAVE and $EDIT definitions identify a WoaS action, while the 's' and 'e' characters identify the associated hotkey.

See the Wikipedia Access key entry for details on how different browsers access these keys.

Custom hotkeys

You can create a custom hotkey associated with a Javascript function that will be called when that hotkey is activated.

Example:
$CUSTOM(myfunction)     k
The myfunction Javascript function will be called (without arguments) when the 'k' hotkey is activated.
