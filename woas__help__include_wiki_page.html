Including the contents of one wiki page inside another wiki page is called transclusion. To accomplish this use an [[Include::]] tag on a line by itself, inserting the name of the page you'd like to transclude after the double colons in the tag.
NOTE: Page names are case sensitive.
Use this code in a page, replacing Page Name with the name of the page you wish to include:
[[Include::Page Name]]
To include the contents of another wiki page in the current page.

Example

Use this code in a page:
[[Include::WoaS::Aliases]]
