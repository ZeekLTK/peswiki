The menu can contain links to pages, wiki interfaces, and shortcuts to custom functions you create. You can edit the menu and sub-menus by clicking the edit icon located in its bottom right corner. Sub-menus belong to Namespaces and are discussed in the sub-menu section there.

If the wiki is erased the menu contents are slightly reduced, with less often used links removed; these links are still accessible from the Special::Advanced page. These are the default contents of the menu with the pages linked briefly explained, with links to more detailed help pages, when available:
  • Main Page - The main page of the wiki.
  • Options - The Options page where basic configuration settings can be set.
  • WoaS History - A page detailing the history of this wiki; this page is deleted if the wiki is erased.
  • WoaS::Help - A page listing all the pages in the WoaS::Help::namespace; opens in the main wiki.
  • WoaS::Aliases - A page that allows you to create replaceable shortcuts used in wiki links.
  • WoaS::Hotkeys - A page that allows you to create key bindings for hotkeys and access keys.
  • WoaS::Plugins - A page that allows you to create, import, and manage plug-ins.
  • Special::All Pages - A page that displays a list of all user-created pages.
  • Go to... - An interface for typing in the title of a page to go to (defaults to the current page).
  • Recent Changes - An optional wiki feature that keeps track of changed pages when enabled.
  • Backlinks - A page that displays a list of pages linking to the current page.
  • New Page - An interface for creating a new page.
  • Duplicate Page - An interface for creating a duplicate of a page (defaults to the current page).
  • Delete Page - An interface for deleting a page (defaults to the current page).
  • Erase Wiki - An interface for erasing the wiki and restoring it to its default state.
  • Search box - A direct-entry interface for entering a search term to search all the user-created pages in the wiki.
