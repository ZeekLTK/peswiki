Importing a WoaS XHTML wiki

Another Wiki on a Stick can be imported into the current wiki by using the Special::Import page.
NOTE: To import a version 0.9.6B or earlier WoaS file you will need a WoaS 0.11.x release as this WoaS cannot import it.
Help pages and the WoaS Javascript code are not imported so it is always best to import an older wiki into a current one as this will also update the older wiki.
  1. Access Special::Import either directly or from Special::Advanced#Maintenance
  2. Click the Browse button and select the wiki you'd like to import
  3. Choose the settings you'd like to use.
    NOTE: If you are importing WoaS < 0.12.0 it is much better to import CSS manually and not automatically because CSS definitions have changed
  4. Click the Import button.
  5. A warning will pop up letting you know that this will overwrite pages with the same title and asking if you're sure.
  6. Click the OK button if you're sure. Otherwise, click the Cancel button.

Importing WSIF

This method is used to import pages that have been exported using the WSIF format (see the WSIF page for more information). Help pages are not imported so it is always best to import WSIF content into a current WoaS file.
  1. Access Special::ImportWSIF either directly or from Special::Advanced#Maintenance
  2. Click the Browse button and select the .wsif file to import
  3. Choose the options to use
  4. Click the Import WSIF button
