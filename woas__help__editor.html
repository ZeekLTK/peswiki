Wiki buttons

Buttons to ease printing wiki syntax.
Wiki Code buttons
ButtonWhat it does
Wiki URLAdd a wiki link to a page.
URLAdd an external link to a page.
=Insert wiki syntax for a level one heading. Use this at the beginning of any line.
==Insert wiki syntax for a level two heading. Use this at the beginning of any line.
ImgInclude a wiki image in the page.
BWrap selected text with bold syntax.
iWrap italic text with bold syntax.
uWrap selected text with underline syntax.
{{{ }}}Wrap wiki syntax around the selected text to make it nowiki (raw) text.
*Insert the wiki syntax for a bulleted list item.
#Insert the wiki syntax for a numbered list item.
@Insert the wiki syntax for an alphanumeric list item.
RulerInsert a ruler (horizontal line). Use this at the beginning of any line.
TagsInterface for inserting tags in your page to categorize it. Use a comma between multiple tags.

Note: it's recommended to use wiki syntax instead of XHTML syntax, because this is a wiki.
