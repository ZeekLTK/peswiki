Wiki links

Below are methods for creating links using wiki syntax.

Anchor link

You can use a special anchor link to link to a heading on the current page. After clicking an anchor link, use the browser's back button to return to where you were on the page.

Use this code in a page, replacing HEADING NAME with the name of the heading you'd like to link to:

[[#HEADING NAME]]

To create an anchor link.

Email link

Use this code in a page, replacing USERNAME with the name of the person, SERVER with the name of their server, and LABEL with the text you'd like displayed as a link:

[[mailto://USERNAME@SERVER|LABEL]]

To create this:
LABEL

Email link with subject

Use this code in a page, replacing USERNAME with the name of the person, SERVER with the name of their server, SUBJECT with the subject of the email, and LABEL with the text you'd like displayed as a link:

[[mailto://USERNAME@SERVER?subject=SUBJECT|LABEL]]

To create this:
LABEL

External link (labeled)

Use this code in a page, replacing URL with the address of the web page, and LABEL with the text for the link:

[[URL|LABEL]]

To create a labeled external link like this one:
Click here for GNU

External link (plain)

Use this code in a page, replacing URL with the address of the web page:

[[URL]]

To create a plain external link like this one:
http://www.gnu.org/

Image as link

Base64 image as link

See the Images page.

Image as link to wiki page

Use this code in a page, replacing IMAGENAME with the name of an embedded image page, and PAGENAME with the name of a page in this wiki:
[[Include::Image::IMAGENAME|style="cursor:pointer" onclick="go_to('PAGENAME')"]]
To create an image that links to a page in this wiki.

Image as link to web page

Use this code in a page, replacing Image::IMAGENAME with the name of an embedded image page, and URL with the address of a web page:
[[Include::Image::IMAGENAME|style="cursor:pointer" border="0" onclick="window.open('URL')"]]
To create an image that links to a web page.

Include link

You can include the contents of another page of this wiki in the current page by using an include link.

Use this code in a page, replacing PAGENAME with the name of the page you'd like to include:
[[Include::PAGENAME]]
To include the contents of another page in the current page.

Internal link (labeled)

Use this code in a page, replacing PAGENAME with the page you want to link to and LABEL with the link text:

[[PAGENAME|LABEL]]

To create a labeled internal link.

Internal link (plain)

Use this code in a page, replacing PAGENAME with the page you want to link to:

[[PAGENAME]]

To create a plain internal link.

HTML links

Below are methods for creating links using HTML syntax.

Alert box link (button)

Use this code in a page, replacing MESSAGE with the content for the alert, and BUTTONTEXT with the text for your button:

<script type="text/javascript">
function alertme () {alert("MESSAGE");}
</script>
<input type=button value="BUTTONTEXT" onClick="alertme()">

To create a button that opens an alert.

Alert box link (labeled)

Use this code in a page, replacing MESSAGE with the content for the alert, and LABEL with the text for your link:

<a href='javascript:alert("MESSAGE");'>LABEL</a>

To create a labeled text link that opens an alert.

Anchor links to anywhere on a page

With a combination of HTML and wiki syntax, you can create anchor links to anywhere on a page, rather than just to headings.

Use this code in a page, replacing NAME with the name of the anchor:

<a name="NAME">

Use this code somewhere else on the same page, replacing NAME with the anchor name:

[[#NAME]]

To create an anchor link to anywhere on a page.

Or use this code somewhere else on the same page, replacing NAME with the anchor name and LABEL with the text for the link:

[[#NAME|LABEL]]

To create a labeled anchor link to anywhere on a page.

Edit page link

Use this code in a page, replacing LABEL with the text for your link:

<a class="woas_link" onclick="woas.ui.edit()" title="Edit this page">LABEL</a>

To create a link that opens the current page in the editor.

External link (button)

Use this code in a page, replacing BUTTONTEXT with the text for your button, and URL with the address of the web page:

<input type="button" value="BUTTONTEXT" onclick="window.open('URL');">

To create a button that opens an external link in another tab or window.

Internal link (button)

Use this code in a page, replacing BUTTONTEXT with the text you want on the button and PAGENAME with the page you want to link to:

<input value="BUTTONTEXT" onclick="woas.go_to('PAGENAME')" type="submit" />

To create a button that opens an internal link in another tab or window.
